HealthyVibes

Participei na produção das telas do projeto (Front-End)

HealthyVibes é um projeto de um aplicativo que permite as pessoas acompanharem suas atividades físicas e hábitos saúdaveis.

O projeto foi feito utilizando Android Studio, Java e MongoDB

HealthyVibes Web

Participei na produção da lógica do site e auxiliando com o banco de dados (Back-end)

HealthyVibes Web é a versão de navegador do aplicativo HealthyVibes com menos funcionalidades comparado ao aplicativo

O Projeto foi feito utilizando TypeScript, NodeJS e MongoDB

AprovaQuiz

Participei desenvolvendo a lógica do aplicativo e site (Back-end)
